## Notifly Flutter Plugin

[![Pub](https://img.shields.io/pub/v/notifly_flutter.svg)](https://pub.dev/packages/notifly_flutter)

[Notifly](https://notifly.tech/) is a versatile messaging service for mobile and web apps. This SDK simplifies the integration of your Flutter iOS and Android apps with Notifly.

### Installation
See the [Setup Guide](https://docs.notifly.tech/developer-guide/client-sdk/flutter-sdk) for setup instructions.
